# Changelog

All notable changes to this project are documented here.
Format based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-07-03

### Added
- Initial release: peer-to-peer real-time tic-tac-toe
- Hand-drawn animated board (SVG stroke-draw animations)
- Room code system for connecting two players
- Unit tests for game logic (Vitest)
- ESLint + Prettier tooling
- CI workflow (lint + test on every push/PR)
- Auto-deploy workflow to GitHub Pages on merge to `main`
