# Contributing

Thanks for considering a contribution! This is a small, free, open-source project — contributions of any size are welcome.

## Getting started

1. Fork the repo and clone your fork.
2. `npm install`
3. `npm run serve` to run locally.
4. Make your changes.
5. `npm run lint && npm test` before committing — CI will run the same checks.
6. Open a pull request against `main` describing what you changed and why.

## Code style

- Formatted with Prettier (`npm run format`) — please run it before committing.
- Keep `src/js/gameLogic.js` free of DOM/networking code so it stays easily testable.
- Small, focused pull requests are easier to review than large ones.

## Reporting bugs / requesting features

Please use the issue templates under **Issues → New Issue**.
